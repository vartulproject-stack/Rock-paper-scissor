# Rock-paper-scissor
Rock Paper Scissors game vs Computer. Click to play, random computer choice, score tracking for player &amp; computer. Win/Lose/Draw animations. Built with HTML, CSS &amp; Vanilla JavaScript.

Day 23/100 of #100DaysOfCode. Rock Paper Scissors game vs Computer. Random computer choice, score tracking, win/lose/draw logic. Animated result display. Built to practice conditional logic, random numbers, and DOM manipulation. Tech: HTML5, CSS3, Vanilla JavaScript, Math.random().
